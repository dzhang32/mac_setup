# New Mac setup

Steps for setting up my MacBook (intel) for personal use and bioinformatics, tested on Big Sur (11.6.1). 

## Applications

### Install HomeBrew

```bash

# from https://brew.sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

```
